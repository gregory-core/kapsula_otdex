# kapsula_otdex
# 🌿 Атмосферный плеер — фон и музыка

Одностраничное веб-приложение, позволяющее переключать расслабляющие фоны и слушать синтезированную атмосферную музыку. Никаких внешних зависимостей, всё работает прямо в браузере.

## ✨ Возможности

- **5 уникальных тем** – Лес, Океан, Космос, Дождь, Город.
- **Живой фон** – плавные градиенты и анимированные звёзды (для космоса).
- **Генеративный звук** – каждая тема создаёт собственное звуковое окружение с помощью Web Audio API (пение птиц, шум волн, эмбиент, дождь, городской гул).
- **Полное управление** – Play/Pause, Mute, регулировка громкости.
- **Горячие клавиши** – быстрое переключение и управление без мыши.
- **Адаптивный дизайн** – удобно на десктопе и мобильных устройствах.

## 🚀 Быстрый старт

1. Скачайте файл `index.html`.
2. Откройте его в любом современном браузере (Chrome, Firefox, Edge, Safari).
3. Наслаждайтесь!

> Для воспроизведения звука требуется взаимодействие пользователя (клик по карточке или кнопке Play).

## 🎮 Управление

| Действие               | Мышь / Сенсор                 | Клавиатура            |
|------------------------|-------------------------------|-----------------------|
| Выбор темы             | Клик по карточке              | `1` – `5`             |
| Воспроизведение / Пауза | Кнопка ▶ / ⏸                | `Пробел`             |
| Выключить звук         | Кнопка 🔊 / 🔇               | `M`                    |
| Громкость              | Ползунок                      | `↑` / `↓`              |
